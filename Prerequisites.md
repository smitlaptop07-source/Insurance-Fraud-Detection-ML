# Prerequisites

## Programming Language
- Python 3.x

## Libraries & Frameworks
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Imbalanced-learn (imblearn)
- XGBoost
- Flask

## Tools & Environment
- Jupyter Notebook / Google Colab
- Git & GitHub

## Concepts Required
- Basic Python programming
- Classification algorithms (Logistic Regression, Decision Tree, Random Forest, XGBoost)
- Data preprocessing (missing values, encoding, scaling)
- Model evaluation metrics (Accuracy, Precision, Recall, F1-Score, ROC-AUC)
- Handling imbalanced datasets (SMOTE)

## Dataset
- Kaggle - Insurance Fraud Detection Dataset (publicly available)
- Link: https://www.kaggle.com/datasets/buntyshah/auto-insurance-claims-data
