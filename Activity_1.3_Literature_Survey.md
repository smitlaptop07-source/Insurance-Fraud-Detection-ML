# Activity 1.3 : Literature Survey

## Existing Research

1. **Logistic Regression** - Used in early fraud detection systems.
   Simple but less effective on complex datasets.

2. **Decision Trees** - Easy to interpret. Used to identify key
   features that indicate fraud.

3. **Random Forest** - Ensemble method that improves accuracy by
   combining multiple decision trees. Widely used in fraud detection.

4. **XGBoost** - Gradient boosting algorithm that gives best
   performance on tabular data. Highly used in Kaggle competitions
   for fraud detection.

5. **Neural Networks** - Deep learning models used for large-scale
   fraud detection but require more data and computing power.

## Gap Identified
Most existing systems use single models. Our project combines
multiple models and selects the best performing one using
evaluation metrics like ROC-AUC and F1-Score.
